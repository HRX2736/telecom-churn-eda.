# Telecom Customer Churn - EDA

This project performs Exploratory Data Analysis (EDA) on a telecom customer churn dataset to understand key factors that influence customer churn.

## Dataset
- File: `telecom_churn.csv`

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Month-to-month contract customers churn the most
- Customers with low tenure (0â€“6 months) have higher churn
- Higher MonthlyCharges are linked with higher churn
- Fiber optic users churn more than DSL in many cases
- Electronic check users churn more compared to auto payments
- Customers without TechSupport churn significantly more
